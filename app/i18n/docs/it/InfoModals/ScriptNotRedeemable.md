# Lo Script non è Riscattabile?

Se si riceve un errore indicando che lo script non è riscattabile quando si tenta di aggiungere il proprio stakepool, si può provare quanto segue:

  1. Ogni conto stakepool creato può essere associato a un solo portafoglio. Se in precedenza si è creato il conto stakepool con un portafoglio diverso (seed diverso), è necessario creare un nuovo conto.

  2. 2. Se in precedenza si è utilizzato un conto di voto per l'acquisto di biglietti, andare alla pagina Conto e creare un nuovo conto. Questo potrebbe ora consentirvi di importare con successo lo script per il vostro stakepool.
